# Sleep Health & Lifestyle Analytics Dashboard

Overview
This repository contains a Power BI project focused on analyzing the relationship between lifestyle factors (stress, physical activity, occupation) and sleep health. The goal was to take a messy dataset, clean it, and build an executive-level dashboard to uncover real insights.
Tech Stack
Tool: Power BI
Data Processing: Power Query (ETL, handling nulls, mean imputation, column splitting)
Calculations: DAX (Custom measures for KPIs)
Key Insights
Occupation Matters: Sales Representatives and Scientists record the lowest average sleep duration, whereas Doctors and Engineers sleep the most.
Stress vs. Sleep:  direct negative correlation was found between high daily stress levels and overall sleep quality.
Sleep Disorders: A significant chunk of the dataset suffers from Insomnia and Sleep Apnea, which directly correlates with higher BMI categories.
Dashboard Features
Dark mode UI for clear, distraction-free viewing.
Custom KPI cards with dynamic formatting.
Interactive slicers for BMI and Gender breakdowns.
